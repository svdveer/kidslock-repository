+# KidsLock Manager
+
+Een lokale Home Assistant Add-on die fungeert als centrale controller voor het KidsLock Android TV ecosysteem.
+
+## Installatie
+
+1. Voeg deze repository toe aan de Home Assistant Add-on Store.
+2. Installeer "KidsLock Manager".
+3. Configureer de IP-adressen van de TV's.
+
+## Functionaliteit
+
+*   **Monitoring:** Ping TV's om status te controleren.
+*   **MQTT:** Integreert met Home Assistant via MQTT discovery.
+*   **Schermtijd:** Beheert dagelijkse limieten en bedtijden.
+*   **Dashboard:** Lokale webinterface voor beheer.
